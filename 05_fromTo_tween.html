Write this:
      animate(logo)
        .during(1)
        .from( {opacity:0.2, left:"400px"} )
        .to( {opacity:0.9, left:"200px"} )
        .thenYoyo(2)
        .easeInOut(Back)
        .start();



Instead of this :
      TweenMax.fromTo(logo, 1, {opacity:0.2, left:"400px"},{opacity:0.9, left:"200px",repeat:2, yoyo:true, ease:Back.easeInOut} );


It knows when to use TweenLite and when to use TweenMax behind the scenes.
